Online Journal for Organ Transplantation

Abstract

The Online Journal for Organ Transplantation (OJOT) is developed mainly for General Hospitals (GH), Clinics and other Health Centers to Manage the Donor Registration and User Maintenance. It is an Online System which only can be access or valid in Kerala state.

The public can retrieve information about the Organ Donation in this Web site. People who interested can register themselves through this system. The application will be processed by administrator and each donor will receive feedback about their application status.

This Online Journal for Organ Transplantation system will help to improve the performance of current situation and overcome the problems that arise nowadays.
